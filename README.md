# data-portfolio

Portafolio web personal con dashboards interactivos de datos.

## Estructura

```
data-portfolio/
├── index.html        ← Página de inicio (presentación + proyectos)
├── dashboard.html    ← Dashboard interactivo con gráficos
├── style.css         ← Estilos globales
├── js/
│   ├── main.js       ← Interactividad general
│   └── charts.js     ← Configuración de gráficos (Chart.js)
└── data/             ← Datos JSON para los dashboards
```

## Tecnologías

- HTML5 + CSS3
- JavaScript (vanilla)
- [Chart.js](https://www.chartjs.org/) para visualizaciones

## Publicación

Este sitio se puede publicar gratis en **GitHub Pages**:
1. Ir a Settings → Pages en el repositorio
2. Seleccionar branch `main` como fuente
3. La URL será: `https://jbendeguz.github.io/data-portfolio`
